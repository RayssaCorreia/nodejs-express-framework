## Express Server  

* install dependencies:
```
npm install
```

* Create a start script on package.json file:
```
  "scripts": {
    "start": "node src/app.js"
  },
```

* run the server
```
npm start
or
node src/server.js
```

---
### 01 First endpoint (Text/plain Hello World)
```
curl -X GET http://localhost:4001/
```
